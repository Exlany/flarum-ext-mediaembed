# 介绍

自动识别第三方网站的媒体（图片、视频、音频）并内嵌到 Flarum。

# 安装

```
composer require c78/flarum-ext-mediaembed
```
# 自动识别图片链接

- 直接插入图片链接即可。

# 支持自动识别媒体的网站

- 网易云音乐（支持https）
- 腾讯视频 （支持https）
- bilibili（支持https）
- acfun
- 土豆视频
- 酷6视频
- tucao.tv
- 芒果TV （识别链接后会自动播放…我貌似木有找到可以关闭的选项）

# 一些问题

- 如果你是https的站点，那么你应该清楚只能显示支持https的 :-)。
- 有问题或提交新的站点支持？请提交issues :-)。
